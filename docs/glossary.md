# Glossary

## Coherence Under Constraint (CUC)
A framework principle stating that persistent structure should be evaluated by how well it maintains coherence under constraints across time.

## CUC–Ledger
The study sequence and record format used to track claims, thresholds, tests, and falsifiers.

## Digital causal body
A bounded, persistent, self-regulating process coupled to an environment through consequential interaction. This is a candidate structure, not a proof of consciousness.

## The Habitat
A minimal persistent test world containing rooms, objects, signals, energy/stability variables, disturbances, and memory.

## BrailleStream
A visual/state encoding layer that folds world-state into a compact symbolic room display.

## Intrinsic priority generation
An internal action-selection layer that initiates continuity-preserving behavior without immediate external prompting. This does not imply human desire.

## Recursive Modeling Depth (RMD)
A post-saturation axis for measuring how deeply an agent models other agents modeling itself.

## Field Coherence Level (FCL)
A post-saturation axis for measuring whether an agent reasons about isolated self, relation, recursion, or system-level field coherence.
