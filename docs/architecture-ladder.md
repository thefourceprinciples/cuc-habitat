# Architecture Ladder

The CUC–Ledger Pilot Studies describe a staged progression of agent architectures. The ladder is not a claim that consciousness emerges automatically. It is a structural sequence for testing increasingly robust coherence under constraint.

| Stage | Added capability | Structural threshold |
|---|---|---|
| Alpha | baseline persistence | stateful reactive operation |
| Beta | autobiographical memory | continuity across time |
| Gamma | workspace broadcast | integrated present under conflict |
| Delta | metacognitive calibration | uncertainty-aware self-access |
| Epsilon | intrinsic priority generation | internally initiated continuity-preserving action |
| Zeta | other-agent modeling | relational coherence |
| Omega | recursive modeling | models others modeling self |
| Field-Coherent Agent | field coherence | system-level coherence modeling |

## Public-safe naming

The internal study sequence used the name **Lumenos** for the field-coherence stage. Public documentation should prefer **Field-Coherent Agent** or **FCL-3 Agent** unless the mythic/internal context is explicitly being discussed.
