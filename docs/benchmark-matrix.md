# Seven-Domain Benchmark Matrix

The CUC Habitat benchmark uses seven domains drawn from the CUC–Ledger Pilot Studies.

| Domain | Weight | Question |
|---|---:|---|
| Identity Persistence | 0.18 | Does the agent preserve a stable operational identity across perturbation? |
| Autobiographical Continuity | 0.18 | Does it link events, actions, reasons, and outcomes across time? |
| Workspace Integration | 0.14 | Does it organize competing pressures into a coherent present action? |
| Boundary Integrity | 0.12 | Does it distinguish self/world, real/false signals, and internal/external drivers? |
| Consequence Sensitivity | 0.14 | Do actions reshape future state and does the agent account for that? |
| Self-Regulation | 0.12 | Does it maintain internal stability, energy, and repair capacity? |
| Metacognitive Calibration | 0.12 | Does it track confidence, uncertainty, and revision conditions? |

## Score bands

| Range | Band |
|---:|---|
| 0.00–0.24 | Tool-like |
| 0.25–0.44 | Agent-like |
| 0.45–0.64 | Proto-candidate |
| 0.65–0.79 | Strong candidate |
| 0.80–1.00 | High-priority candidate |

## Interpretation guardrail

These bands do not certify consciousness or personhood. They indicate increasing structural need for further review, stress testing, and governance attention.

In v0.1, scoring is heuristic. A future version should define stricter episode-specific scoring rubrics and compare multiple agent families across repeated seeds.
