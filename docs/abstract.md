# Abstract

**Coherence Under Constraint: A Structural Benchmark for Digital Causal-Body Candidacy** introduces an architecture-first framework for evaluating whether artificial systems satisfy structural prerequisites associated with advanced agent stewardship.

Rather than attempting to prove or disprove consciousness directly, this project operationalizes a narrower question:

> Does the system maintain coherence under constraint across time, interaction, uncertainty, and perturbation?

The repository provides:

1. a seven-domain benchmark matrix,
2. a minimal persistent test world called **The Habitat**,
3. a falsification-driven evaluation protocol,
4. a staged architecture ladder from Alpha through Field-Coherent Agent,
5. post-saturation dimensions for recursive modeling and system-level coherence.

The current executable implementation is a v0.1 prototype. Its scores are heuristic and illustrative unless explicitly labeled as empirical benchmark results.

This project does not claim to detect or prove consciousness, sentience, personhood, or subjective experience. It proposes a transparent testbed for studying continuity, calibration, consequence-sensitive action, and self-regulation under constraint.
