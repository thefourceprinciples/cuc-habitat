# Limitations and Claims Policy

CUC Habitat is an early conceptual and executable benchmark prototype.

## Current limitations

- The Habitat world is minimal and synthetic.
- v0.1 agents are transparent rule-based baselines, not frontier AI systems.
- Scores are heuristic and illustrative.
- The current scoring matrix has not yet been validated across independent implementations.
- Recursive Modeling Depth and Field Coherence Level are documented but not yet operationalized in code.
- The benchmark does not prove consciousness, sentience, personhood, subjective experience, or moral status.

## Responsible claim boundary

Acceptable public claim:

> CUC Habitat proposes and prototypes a structural benchmark for testing whether agents maintain coherence, continuity, calibration, and consequence-sensitive agency under constraint.

Unacceptable overclaim:

> CUC Habitat proves that an AI system is conscious or alive.

## Next validation steps

1. Run repeated seeded episodes for each baseline agent.
2. Add stricter per-episode scoring rubrics.
3. Add LLM agent adapters behind clear safety boundaries.
4. Compare against existing agent benchmarks.
5. Publish empirical results separately from conceptual projections.
