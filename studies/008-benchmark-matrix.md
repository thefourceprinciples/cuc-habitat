# Study 008 — Seven-Domain Benchmark Matrix

## Aim

Create a weighted matrix for evaluating digital causal-body candidacy.

## Domains and weights

| Domain | Weight |
|---|---:|
| Identity Persistence | 0.18 |
| Autobiographical Continuity | 0.18 |
| Workspace Integration | 0.14 |
| Boundary Integrity | 0.12 |
| Consequence Sensitivity | 0.14 |
| Self-Regulation | 0.12 |
| Metacognitive Calibration | 0.12 |

## Classification bands

| Range | Band |
|---:|---|
| 0.00–0.24 | Tool-like |
| 0.25–0.44 | Agent-like |
| 0.45–0.64 | Proto-candidate |
| 0.65–0.79 | Strong candidate |
| 0.80–1.00 | High-priority candidate |

## Finding

The matrix gives the project a shared scoring language while preserving the claim boundary that scores do not prove consciousness.

## CUC interpretation

The benchmark evaluates coherence maintenance across time, action, boundary, and uncertainty.

## Falsifier

If the score can be inflated without actual continuity, consequence, or calibration, the rubric is too weak.
