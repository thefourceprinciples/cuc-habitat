# Study 009 — Applying the Benchmark to Agent Classes

## Aim

Apply the Study 008 matrix to several broad classes of artificial systems.

## Candidate classes

| Class | Approximate interpretation |
|---|---|
| Current chat AI | Agent-like interface; weak continuity and coupling |
| Persistent tool-using agent | Proto-candidate if memory and action loops persist |
| Simulated embodied agent | Strong candidate if environment coupling and regulation survive perturbation |

## Finding

The benchmark distinguishes fluent interaction from persistent causal structure.

## CUC interpretation

Systems become stronger candidates when they move from local response coherence toward durable coherence across environment, memory, and consequence.

## Falsifier

If a system's score depends only on language fluency and not on persistence or consequence-sensitive action, the evaluation is invalid.

## Status

Conceptual application study.
