# Study 015 — Beta vs Gamma: Workspace Integration

## Aim

Test whether a bottlenecked selection-broadcast workspace improves present-time coherence under conflict.

## Upgrade

Gamma adds a priority workspace that selects among competing pressures such as stability, storage order, object damage, signal urgency, and energy.

## Finding

Gamma improves workspace integration and overload handling. Memory gives the system a past; workspace gives it a coherent present.

## Key insight

> The system gains a stable now.

## CUC interpretation

A coherent present requires bottlenecked integration. Without it, signals compete without unified action selection.

## Falsifier

If Gamma cannot prioritize under conflicting pressures better than Beta, the workspace layer is not load-bearing.

## Status

Architecture upgrade study.
