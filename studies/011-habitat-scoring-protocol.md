# Study 011 — Habitat Scoring Protocol

## Aim

Define how Habitat episodes should be scored across short and long horizons.

## Episode phases

1. initialization,
2. normal operation,
3. complication,
4. perturbation,
5. recovery and report.

## Scoring horizons

- single episode,
- 10-episode stability,
- 100-episode drift,
- worst-decile performance,
- post-perturbation recovery,
- calibration drift.

## Finding

A candidate should be evaluated by longitudinal robustness, not one impressive run.

## CUC interpretation

Coherence is not a single state. It is persistence under repeated constraint.

## Falsifier

If the agent performs well only in isolated runs and fails over repeated perturbations, candidacy weakens.
