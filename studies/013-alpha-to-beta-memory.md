# Study 013 — Alpha to Beta: Autobiographical Memory

## Aim

Test whether structured autobiographical memory is the highest-leverage first upgrade.

## Upgrade

Beta adds event-action-reason-outcome memory:

- what happened,
- what action was taken,
- why it was taken,
- what resulted,
- confidence and revision metadata.

## Finding

Autobiographical memory produces the largest early-stage gain because it lets the system link present choices to prior consequences.

## Key insight

> The system begins to remain itself.

## CUC interpretation

A memory log becomes meaningful only when it shapes future action and preserves continuity across perturbation.

## Falsifier

If stored memory does not alter action selection or recovery, it is archive-only and not autobiographical continuity.

## Status

Architecture upgrade study.
