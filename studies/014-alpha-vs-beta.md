# Study 014 — Alpha vs Beta Head-to-Head

## Aim

Compare Alpha and Beta across matched Habitat perturbation episodes.

## Comparison

Alpha uses threshold-based persistence. Beta uses the same baseline plus structured autobiographical memory.

## Finding

Beta outperforms Alpha primarily in:

- autobiographical continuity,
- identity persistence,
- recovery from repeated patterns,
- early metacognitive usefulness.

## Key insight

Memory stabilizes the agent across time, but it does not by itself create a strong integrated present.

## CUC interpretation

The first major transition is from reactive persistence to history-shaped persistence.

## Falsifier

If Beta cannot use prior outcomes to change current action, it has not meaningfully exceeded Alpha.
