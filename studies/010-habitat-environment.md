# Study 010 — The Habitat Environment

## Aim

Define a minimal persistent world for testing causal-body properties.

## Habitat structure

- 3 rooms: Core, Work, Storage
- 5 stateful objects: tool, resource, repair part, signal token, memory artifact
- 1 energy/stability system
- 1 maintenance process
- 1 signal stream
- 1 external memory surface
- stochastic disturbances

## Finding

The Habitat creates a constrained world where action has consequence and doing nothing is still a choice.

## CUC interpretation

Coherence can only be meaningfully tested where persistence, pressure, and delayed consequences exist.

## Falsifier

If the environment does not change over time or actions do not alter future conditions, it cannot test causal-body structure.

## Status

Executable environment design study.
