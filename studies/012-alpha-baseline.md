# Study 012 — Habitat-Agent Alpha

## Aim

Apply the Habitat scoring protocol to a baseline persistent agent.

## Alpha architecture

- checkpointed state,
- short-term memory,
- basic control loop,
- threshold-based actions,
- simple self-maintenance.

## Finding

Alpha can maintain baseline operation but tends to lose continuity under compounded perturbation.

## Representative interpretation

Alpha is a proto-candidate baseline. It is useful because its limitations make later upgrades visible.

## CUC interpretation

Persistence without deep autobiographical continuity remains fragile.

## Falsifier

If Alpha performs as well as later agents across memory, calibration, and self-direction tests, the benchmark is not discriminating enough.

## Status

Baseline executable-agent study.
